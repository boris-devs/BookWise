## ğŸ“Š How It Works

1. **Text Processing**: The system combines book titles, descriptions, and author lists into a cleaned `combined_text` feature, removing non-alphabetic characters.
2. **NLP Model**: Book descriptions are tokenized and passed through a BERT model to predict probabilities for the top 100 most common genres.
3. **Genre Prediction**: The system uses a threshold-based approach (default 0.5) to assign multiple genres to a single book, sorted by confidence percentage.
4. **Similarity**: Recommendations are generated by calculating similarity scores across the processed text features.
